# Mokkosuの始め方

以下のページからインストーラがダウンロードできます。

* [リリースページ](https://github.com/lambdataro/Mokkosu/releases)

現時点でドキュメント・サンプルがともに不足していますが、
なるべく早くこの状況を改善したいと考えています。

現時点でMokkosuを始めてくださる方は、
以下のことをするとよいと思います。
 
1. 標準ライブラリのソースコードを読む
    * [Stdlib.mok](https://github.com/lambdataro/Mokkosu/blob/master/VS2013/MokkosuCore/Stdlib.mok)
    * [Print.mok](https://github.com/lambdataro/Mokkosu/blob/master/VS2013/MokkosuCore/Print.mok)
    * [Color.mok](https://github.com/lambdataro/Mokkosu/blob/master/VS2013/MokkosuCore/Color.mok)
    * [2D.mok](https://github.com/lambdataro/Mokkosu/blob/master/VS2013/MokkosuCore/2D.mok)
2. [Sampleディレクトリ](https://github.com/lambdataro/Mokkosu/tree/master/Sample)の
ソースコードを読む。
3. リファレンスマニュアルを読む。
